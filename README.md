# HealthGrid ( A Hospital Inventory System )

A comprehensive hospital inventory management system built with C# and Windows Forms.

## Features

- **Inventory Management:** Keep track of medical supplies, equipment, and medicines.
- **Stock Tracking:** Manage stock levels within hospital departments.
- **User Interface:** Simple, easy-to-use Windows Forms interface.
- **Database Integration:** Scalable local storage for reliable data persistence.

## Getting Started

### Prerequisites

- [Visual Studio](https://visualstudio.microsoft.com/) 2019/2022 (with .NET Desktop Development workload).
- .NET Framework.

### Running the Application

1. Clone the repository: `git clone https://github.com/Sibghat-629/HealthGrid.git`
2. Open the solution file `Hospitalinventorysystem.sln` in Visual Studio.
3. Build the solution (Ctrl + Shift + B).
4. Run the application (F5).
